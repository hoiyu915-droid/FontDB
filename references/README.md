# Visual references

Register each source as metadata, not as a font.

Suggested record:

```yaml
id: REF_0001
source_type: montage
source_file: null
captured_at: 2026-07-11
language: zh-Hant
range: 1-100
purpose: archetype_and_treatment_inspiration
font_identity_claim: prohibited
notes:
  - labels are descriptive marketing names
  - many examples combine ordinary glyphs with post-processing
```

The initial 100-style montage should later be decomposed into:

- likely archetype
- functional role
- treatment recipe
- legibility risk
- closest existing FontDB profile
- unresolved visual features

Keep original images outside the production catalog until provenance and reuse rights are known.
